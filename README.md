# 🛡️ Fantasy-Marketplace-Frontend

Uma interface de e-commerce imersiva para RPGs, desenvolvida com foco em tecnologias modernas de frontend e design responsivo.

## ✨ Funcionalidades

-   **Sistema de Ouro:** Inicie sua jornada definindo a quantidade de moedas de ouro que o aventureiro possui.
-   **Navegação por Categorias:** Filtre facilmente entre Alimentos, Poções, Armas e Armaduras.
-   **Busca em Tempo Real:** Encontre rapidamente itens mágicos através do campo de pesquisa.
-   **Carrinho de Compras Interativo:** Gerencie seus itens antes de finalizar a compra, com indicação visual caso o ouro seja insuficiente.
-   **Lista de Desejos (Wishlist):** Salve itens raros para compras futuras.
-   **Feedback Visual Dinâmico:** Notificações (toasts) para cada ação realizada na loja.
-   **Persistência de Dados:** Seus itens no carrinho, lista de desejos e saldo de ouro são salvos localmente (LocalStorage), para que você nunca perca seu progresso ao recarregar a página.

## 🎨 Design & Estética

O projeto utiliza uma estética de **Glassmorphism**, combinando:
-   Filtros de desfoque (`backdrop-filter`) para um visual de vidro translúcido.
-   Tipografia clássica (**Cinzel**) para títulos e moderna (**Inter**) para leitura.
-   Ícones estilizados do **Font Awesome**.
-   Layout totalmente responsivo para dispositivos móveis e desktop.

## 🛠️ Tecnologias Utilizadas

-   **HTML5** - Estrutura semântica.
-   **CSS3** - Estilização moderna com variáveis (Custom Properties) e Flexbox/Grid.
-   **JavaScript (Vanilla)** - Lógica de estado da aplicação, manipulação de DOM e persistência local.
-   **Google Fonts** - Fontes customizadas.
-   **Font Awesome** - Biblioteca de ícones.

## 🚀 Como Executar

1.  Clone este repositório:
    ```bash
    git clone https://github.com/BOliveiraBarbosa/fantasy-marketplace-frontend.git
    ```
2.  Abra o arquivo `index.html` em qualquer navegador moderno.

## 📂 Estrutura de Arquivos

-   `index.html`: Estrutura principal da página.
-   `style.css`: Estilos e design do projeto.
-   `script.js`: Lógica de gerenciamento de itens, carrinho e estado.
-   `/assets`: Imagens e ícones dos itens e fundo.
